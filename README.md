# LING 250 Skills

Tutoring skills for Northwestern's LING 250: Introduction to Phonetics &
Phonology. Part of the NUCS338 skills-scaffolding project.

Currently five skills covering HW1 (Consonant Articulation & IPA):
- `validate-prereq` — probes student's pre-knowledge of foundational concepts
- `dx-prereq-gaps` — locates specific prerequisite gaps
- `repair-miscons` — repairs common phonetics misconceptions Socratically
- `escalate-hint-lvl` — graduated hint ladders for stuck moments
- `contrastive-hint` — distinguishes confused consonant pairs by feature

See `assignments/hw1.md` for the assignment spec and
`context/misconceptions.md` for the misconceptions catalog.
